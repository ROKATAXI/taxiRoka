# taxiRoka
## 프로젝트 개요 및 셋업
- python 3.11
- 가상환경 세팅후 requirements.txt 를 통해 필요 라이브러리 설치 (추가적으로 사용하는 라이브러리가 생길 경우 반드시 추가할 것)
  - `pip install -r requirements.txt`

## 프로젝트 설명 및 주요 기능
- [기획안 링크](https://www.figma.com/file/7moP30bEJTYJeyoaSUACyt/PIRO19-ROKA?type=design&node-id=0-1&mode=design)

## 기능 상세
- 회원 관련 기능
  - 회원가입
  - 로그인
  - 회원 정보 수정
  - 회원탈퇴
- 채팅방 관련 기능
  - 채팅방 목록
  - 채팅방 생성 (출발지점, 도착지점, 예약일시, 최대 인원, 좌석 위치 설정 필요)
  - 채팅 입장 및 퇴장
  - 유저간 채팅 
- 캘린더 관련 기능
  - 휴가 일정 등록
  - 휴가 일정 수정
  - 휴가 일정 삭제
